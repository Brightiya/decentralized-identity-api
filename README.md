# Decentralized Identity API (monorepo)

This repository contains a starter implementation for an SSI-based Identity & Profile Management API:

- `contracts/` — Solidity IdentityRegistry contract (ERC-725 like)
- `backend/` — Express.js API (DID, profile, VC)
- `frontend/` — Angular app (placeholder)

See `backend/README-backend.md` for backend usage.
